# TaskPro
Sistema simples de gerenciamento de tarefas diárias. Permite adicionar, marcar como concluída e remover tarefas. Desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnologias

- HTML
- CSS
- JavaScript

## Como usar

Basta abrir o arquivo `index.html` no navegador. Nenhuma instalação necessária.
